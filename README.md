# baseballBoard
A simple dashboard to show the score of a baseball game. The score is shared between all users. To change the score, just click on the elements.

![Image of the BaseballBoard](https://raw.githubusercontent.com/derbl4ck/derbl4ck.github.com/master/baseballBoard.PNG)

## Standalone
You can start using the dashboard without any dependencies. Just select the executable suiting your platform in the `builds` directory and navigate to port 3030 in your browser.

## Installation via Source Code

  - run `npm install`
  - now you can start the server using `npm run start` or `node app.js`
  - clients can navigate their web browser to port 3030 of your host machine
